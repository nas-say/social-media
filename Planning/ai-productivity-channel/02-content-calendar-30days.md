## Adjustments After Week 1

### If Low Views (<1,000 avg):
- Improve hooks (first 1 second)
- Test different posting times
- Use more trending audio
- Engage more with similar accounts

### If Low Engagement (<5%):
- Add clearer CTAs
- Make content more actionable
- Ask questions in captions
- Create more "save-worthy" content

### If Low Saves:
- Create list/tutorial content
- Add "save this for later" CTA
- Make content reference-worthy
- Include templates/prompts
