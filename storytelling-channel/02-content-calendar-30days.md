## Content Creation Workflow

### Research Phase (Sunday, 2-3 hours)

**Tools**:
- Wikipedia (starting point only, not final source)
- Britannica, History.com (credible general sources)
- Academic databases (if accessible)
- Reddit r/UnresolvedMysteries, r/AskHistorians (for story ideas)
- Newspaper archives (for true crime)

**Process**:
1. Find 30 story ideas (browse sources above)
2. Verify each story has at least 2 credible sources
3. Save sources for each story
4. Organize in Notion/Google Doc

**Template**:
```
Story Title: [Name]
Category: [Historical/Crime/Mystery/etc.]
Hook: [One sentence opener]
Key Facts:
- Fact 1
- Fact 2  
- Fact 3
Twist/Climax: [The dramatic reveal]
Sources:
- [Link 1]
- [Link 2]
```

---

### Scripting Phase (Monday, 2-3 hours)

**Use ChatGPT to Draft**:
```
Prompt: "Write a 60-second story script about [topic]. 
Structure: Hook (dramatic opener), Setup (context), Rising action 
(build tension), Twist (surprising turn), Resolution (payoff), 
Final hook (encourage engagement). Tone: mysterious, compelling, 
factual. Use short sentences. Target: 150-180 words."
```

**Then Edit**:
- Verify all facts
- Add dramatic pauses
- Include CTA at end
- Ensure cliffhanger if multi-part

**Save all scripts in one document with clear naming**:
- Day1_AM_LibraryAlexandria.txt
- Day1_PM_EvilNurse.txt
- etc.

---

### Creation Phase (Tuesday, 4-5 hours)

**Batch create all 21 reels for the week**:

**Hour 1: Gather Assets**
- Download stock footage for all stories
- Organize in folders by story
- Queue up music tracks

**Hour 2-4: Create in Canva**
- Use templates (see 04-content-templates.md)
- Overlay text from scripts
- Add footage/images
- Time to music
- Export each as MP4

**Hour 5: Final Touches**
- Add AI voiceover (if using)
- Check quality on phone
- Create backup versions

1. Upload all 21 reels to Meta Business Suite
2. Write captions:
   - Story title

   - CTA

3. Schedule for optimal times

## Script Templates by Category

### True Crime Template:
```
[HOOK: Shocking opener about the criminal/crime]

[SETUP: Who they were, seemed normal]

[RISING ACTION: The crime begins, escalates]

[DISCOVERY: How they got caught / pattern noticed]

[REVEAL: The shocking truth / extent of crimes]

[RESOLUTION: Current status / aftermath]

[ENGAGEMENT: Thought-provoking question or series tease]
```

### Historical Mystery Template:
```
[HOOK: The mystery in one sentence]

[CONTEXT: When/where it happened]

[THE EVENT: What happened / what disappeared / what was found]

[INVESTIGATION: Initial theories / what they discovered]

[TWIST: New evidence / surprising fact unknown to most]

[CURRENT STATUS: Is it still unsolved? Recent developments?]

[ENGAGEMENT: What do YOU think happened?]
```

### Survival Story Template:
```
[HOOK: The desperate situation in vivid detail]

[SETUP: How they got there]

[CRISIS: Moment it all went wrong]

[SURVIVAL TACTICS: Specific things they did (numbered list works well)]

[TURNING POINT: The moment they were saved / saved themselves]

[AFTERMATH: Where are they now / lasting impact]

[ENGAGEMENT: Could you survive this? Comment below]
```
