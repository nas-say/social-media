## üÜò Troubleshooting Common Issues

### "My videos look low quality on Instagram"
**Fix**: 
- Export from Canva at 1080p (minimum)
- Upload via Wi-Fi, not cellular
- Use .MP4 format
- Keep file under 200MB

### "My scheduled posts aren't posting"
**Fix**:
- Verify account is connected in Meta Suite
- Check if you have "Professional Account" enabled
- Try rescheduling 5 min later

### "Can't find trending audio"
**Fix**:
- Use Instagram app (not Meta Suite) for posting
- Browse Reels, save audio you like
- Check "Audio" tab in Reels camera

### "Canva templates not working on mobile"
**Fix**:
- Create on desktop, edit on mobile
- Or use Canva mobile app templates
- Save as "Brand Template" for easy access

## üí° Tool Hacks & Pro Tips

### Canva Hacks:
1. **Brand Kit**: Set up once, maintains consistency
2. **Magic Resize**: Create once, resize for all platforms
3. **Templates**: Build library of 10-15 reusable templates
4. **Video Speed**: Use speed controls (1.5x for boring parts)
5. **Folders**: Organize by content pillar

### ChatGPT Hacks:
1. **Custom Instructions**: Set your tone/style once
2. **Saved Prompts**: Keep best prompts in Notion
3. **Batch Requests**: Generate 30 hooks at once
4. **Repurpose**: Ask it to reformat content for different platforms

### Meta Suite Hacks:
1. **Drafts**: Save reels as drafts if not ready
2. **First Comment**: Schedule your hashtags as first comment
3. **Analytics Dashboard**: Check weekly, not daily
4. **Cross-posting**: Auto-post to Facebook for reach
