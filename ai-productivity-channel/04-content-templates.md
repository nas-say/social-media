## ğŸ¬ Animation Timeline Standards

### Optimal Reel Length: 7-15 seconds
- 7 sec: Quick tips, lists
- 10 sec: Tutorials, comparisons
- 15 sec: Detailed walkthroughs

### Hook (0-1 second): CRITICAL
- Text must be readable instantly
- Use pattern interrupt
- Bold, contrasting colors

### Content (1-12 seconds):
- Change scene every 2-3 seconds
- Keep text on screen long enough to read (3 sec rule)
- Use progress indicators for lists

### CTA (last 2-3 seconds):
- Clear action: Follow, Save, Click bio
- Make it visual (arrow, button effect)
- Don't let it feel forced

### Color Palette:

**Primary Colors**:
```
Deep Navy:    #1A1F3A (backgrounds)

```

**Accent Colors**:
```
Electric Purple: #8B5CF6 (highlights, buttons)

```

**UI Colors**:
```
Success Green:   #22C55E
Warning Yellow:  #EAB308
Error Red:       #EF4444
White:           #FFFFFF
Gray:            #94A3B8 (secondary text)
```

### Typography Scale:

```
Hero/Hook:       80-100pt (Montserrat Black)
Title:           64-72pt (Montserrat Bold)
Heading:         48-56pt (Montserrat Semibold)
Body:            36-44pt (Inter Regular)
Caption:         28-32pt (Inter Light)
```

**Font Pairings**:
- Headlines: Montserrat (modern, bold, tech feel)
- Body: Inter (readable, professional)
- Code/Technical: JetBrains Mono (for prompts)
- Quotes: Playfair Display (elegant, serif)

### Spacing Standards:

- Padding: 40px minimum from edges
- Line height: 1.4x for readability
- Letter spacing: -0.02em for large text
- Element spacing: 24px minimum

## ğŸ”„ Batch Creation Workflow

### Sunday Session (3 hours):

**Hour 1: Prep**
1. Open calendar (02-content-calendar-30days.md)
2. Queue up 14 scripts for the week
3. Gather any screenshots/assets needed

**Hour 2: Create in Canva**
1. Duplicate appropriate template
2. Customize text for Reel #1
3. Adjust colors/animations if needed
4. Export as MP4
5. Repeat for all 14 reels

**Hour 3: Schedule**
1. Upload all to Meta Business Suite
2. Write captions + hashtags
3. Schedule for optimal times
4. Set phone reminders for engagement

### Pro Tips:
- Create templates once, reuse forever
- Save common elements as Canva components
- Use Canva's "Brand Kit" to stay consistent
- Batch similar content types together
- Leave room for trending sounds (swap audio later)

## ğŸ“± Mobile vs. Desktop Creation

### Mobile (Canva App):
**Pros**:
- Create anywhere
- Easier to preview how it looks on phone
- Quick edits on the go

**Cons**:
- Smaller screen = harder precision
- Fewer advanced features
- Slower for batch work

### Desktop (Canva Web):
**Pros**:
- Faster workflow
- Better for batch creation
- More control
- Easier copy/paste

**Cons**:
- Need computer access
- Can't preview mobile view as easily

**Recommendation**: Create on desktop, preview on mobile before posting.
